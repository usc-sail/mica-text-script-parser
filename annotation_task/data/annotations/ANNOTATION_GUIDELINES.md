## Annotation Guidelines for Screenplay

There are 40 sheets in the annotation workbook, each containing 250 non empty lines of a movie screenplay. The task is to annotate each line by copying it into a cell to the right under the appropriate column in the same row.

The first sheet has already been done for you to understand how the annotation is to be done.

There are seven columns, each corresponding to a class that a screenplay line can belong to. Following is the definition and one example for each class -

1. #### Scene Boundary / Heading (S)

    A line is a scene boundary / heading if it starts a new scene. A scene is a continuous set of events which occurs in the same location at a particular time in the screenplay. Often a new scene will start with a heading that contains keywords like __INT__ (interior) and __EXT__ (exterior) and provides information about the location and time.

        EXT. LARGE TERRACE/WASHINGTON D.C. - EVENING

2. #### Scene Description / Direction (N)

    A line is a scene description / direction if, simply put, it describes what is happening in the scene. Most of the lines which are not dialogues, character names and scene boundaries, fall under scene description.

        A major fund raising party is under way. The setting is
        spectacular. A terrace overlooking the Washington Mall and
        the Capitol Building.
        
        ADRIAN HELMSLEY, 32, stands with a group of young POLITICAL
        AIDES. He is the only African-American among them.
        
        One of the aides spots CARL ANHEUSER, 58, White House Chief
        of Staff, working the crowd.

    All the lines here should be copied under N.

3. #### Character (C)

    A line is a character if it includes the character name and the lines following it includes the character's utterance. Sometimes the line might also contain information like __VO__ (voice over) and __OS__ (off screen) in parantheses following the character name.

4. #### Dialogue (D)

    A line is a dialogue if it includes some character's utterance. A __Character__ line is always followed by __Dialogue__ lines.

                    ADRIAN
        I still can't believe that Wilson chose
        him of all people to run the White
        House.

    Here, the first line should be copied under C, and the following lines should be copied under D.

5. #### Dialogue Metadata (E)

    A line is a dialogue metadata line if it occurs within dialogue lines and is not part of the speaking character's utterances, but instead providing additional information while the character is speaking. It often appears between parantheses.

                    HARLIE FROST (CONT'D)
        Pretty neat, huh? It's a big hit on
        YouTube. I did the animation myself.
                    (PAUSE)
        I know what you're thinking. But all
        the governments know about this.

    Here, the first line is a Character line and should be copied under C. The line containing (PAUSE) is a dialogue metadata line, and should be copied under E. The rest should be copied under D.

    Character, Dialogue and Dialogue Metadata (if any) lines __always__ occur together. 

6. #### Transition (T)

    A line is a transition if it contains transition elements like _FADE IN, FADE OUT, CUT TO_ which tells how one scene links to the other.

        Prof. West slowly steps closer. When he discovers that the
        water in the tank below is boiling, his face goes pale.
          
                                                        CUT TO:
          
        EXT. LARGE TERRACE/WASHINGTON D.C. - EVENING
          
        A major fund raising party is under way. The setting is

    The line containing CUT TO: should be copied under T.

7. #### Metadata (M)

    A line which describes any metadata of the screenplay and has no link to the screenplay content is a metadata line. They often appear in the screenplay header and footer. Do not include noise like page numbers, continued keywords or empty lines in metadata.

        2012
                Written by
          
                Roland Emmerich & Harald Kloser

                                                Second Draft
                                                February 19th, 2008

    These are all metadata lines.


If the annotator believes that a particular line doesn't fall under any of the above categories or they choose to make some comment, please use the remarks column to express your suggestions. The aforementioned classes have been decided following the nomenclature of existing work and the book [Hollywood Standard 2nd Ed.](https://www.amazon.com/Hollywood-Standard-Complete-Authoritative-Script/dp/1932907637).
